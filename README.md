# HRMS (Human Resource Management System)

A full-stack web application for managing human resources built using the **MERN stack** (MongoDB, Express.js, React, Node.js).  

This system allows organizations to manage employees, attendance, payroll, and internal communications efficiently.

- Real-time chat using **Socket.IO**
- Database: **MongoDB**

### Frontend
- Built with **React** and **Tailwind CSS**
- Responsive design
- Employee dashboards
- Forms for adding/editing employees
- Notifications & alerts using **React Toastify**
- Integration with backend APIs
